# Carpeta de Imágenes de Santos

Esta carpeta contiene las imágenes de los santos para el calendario.

## Formato de nombres de archivo
Los nombres de archivo deben coincidir con los especificados en `santos.csv`.

Ejemplo:
- `san_vicente_martir.jpg`
- `santa_emerenciana.jpg`
- `san_ildefonso.jpg`

## Cómo agregar imágenes

1. Descarga las imágenes de los santos desde:
   - Wikimedia Commons: https://commons.wikimedia.org
   - Vatican News: https://www.vaticannews.va/es/santos.html
   - Otras fuentes de dominio público

2. Renombra el archivo según el nombre especificado en `santos.csv`

3. Guarda el archivo en esta carpeta

4. Formatos recomendados: JPG, PNG
5. Tamaño recomendado: 300x300px o superior

## Imagen por defecto

Si no hay imagen disponible, la web mostrará un símbolo de cruz (✝) como placeholder.

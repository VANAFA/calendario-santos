# Carpeta Web

Esta carpeta contiene los archivos fuente del sitio web.

**NOTA**: Para GitHub Pages, los archivos HTML se copian a la raíz del proyecto automáticamente.

## Estructura

- `index.html` - Página principal del calendario
- `cita-biblica.html` - Página del evangelio del día  
- `images/` - Imágenes de santos (1000+)

## ⚠️ Importante

Los archivos HTML en la **raíz del proyecto** son los que GitHub Pages sirve.
Esta carpeta contiene las **versiones de desarrollo** de los archivos.

Cuando hagas cambios:
1. Edita los archivos en `web/`
2. Copia a la raíz: `cp web/*.html .`
3. Commit y push
